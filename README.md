Expense Tracker

A modern open-source expense tracker that helps users record transactions, categorize expenses, monitor spending habits, generate financial summaries, and improve personal budgeting through a simple and intuitive interface.

Features

- Add expenses
- Store expense records
- Simple and beginner-friendly design
- Open-source and community-driven

Installation

1. Clone the repository
2. Run the Python script

python expense_tracker.py

Roadmap

Version 1

- Add expenses
- Save expenses to a file

Version 2

- View all expenses
- Calculate total spending

Version 3

- Expense categories
- Monthly reports

Version 4

- Budget tracking
- Data export
